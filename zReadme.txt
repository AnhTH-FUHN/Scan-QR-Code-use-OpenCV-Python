1, Install Python 3.7
. Windows download + setup local tu trang chu cua python
. Lunix: sudo apt-get install python3.7
2, Cai pip cho python 3.7
. Windows + Lunix: python get-pip.py
3, Cai OpenCV cho python
. Windows + Lunix: pip install cv2
4, Cai cac thu vien chuong trinh yeu cau theo cu phap
. Windows + Lunix: pip install <ten thu vien>
